---説明---

・外部親設定
任意のモデルの頭ボーン辺りを外部親登録する。
その後に位置を微調整する。

・モデル描画順設定
透過の関係上、極力後ろの方に配置する。
透過先のエッジは見えなくなる。

・モーフ
変形は全てモーフで行う。
数量%は元の大きさから。

規模：全体の縮尺を変動。
x：横の縮尺を変動。
y：高さの縮尺を変動。
z：奥行の縮尺を変動。
x(下側)：アクセの下側に限り横の縮尺を変動。
z(鼻)：アクセの真中付近に限り奥行の縮尺を変動。

U：テクスチャを移動させる(アクセの色彩を変更)。
V：テクスチャの描画領域を狭くする。

CLEAR：透明度を上げる
